Event stamp 1 solution

From: Carolyn Stearns | Date: 2004-01-24 13:16:17 UTC
Here is what I have done with a couple event stamps. I carve the
stamp for the event and everyone there gets the image. I then carve
out a portion like the date of the event or some piece and the
revised stamp is issued as a box or HH. Then the event one is unique
and the rest of the LB workd can have a piece of the fun. This might
not work with every image but I have chosen some ahead because it
would work well. Leader of the Pack
