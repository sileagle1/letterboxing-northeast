The way I did mine in Andover, MA is to hollow out a
reference book that is ready to be retired from
circulation and make it a letterbook.

Another way is to have the box under the check out counter
and they have to ask for it. Depending on how the shelves
are designed in the stacks, you might velcro a microbox
under a shelf.

I have a library box (in Enfield, CT). We met with the librarian, who

Re: Library boxes? (how to?)

From: Warrior Woman | Date: 2006-03-29 17:25:56 UTC
I've only found one library box, and that was the Phantom Phiddler's
Kingston Free Library in RI near URI.

It is a wooden box that is shaped to look just like a hardcover
book. (You could probably do it with a hollowed out hardcover book)
It has its own dewey decimal number on it. It looks just like any
other book on the shelf.

An ordinary library patron would probably just skip right by it. A
letterboxer will find it immediately once the clues are solved.

Since it is a normal sized "book", and it has its own dewey decimal
number on it, any reorganization of the library would not matter. It
would stay in the same sequence with the other books, and any
additions or deletions to the library's contents also would not
affect the ability to find the box.

Hope this info helps!

Warrior Woman

Re: [LbNE] Library boxes? (how to?)

From: dave & diane | Date: 2006-03-30 05:31:56 UTC
This was actually Dave's project, but I made the boxes. We used the ready-to-finish book-boxes from a craft store. I covered it with muslin and we printed up labels. We used "Modge-Podge" to attach the fabric and labels. Dave then painted the sides of the book to look like the page edges to complete the faux book. Then Dave met with the local librarian to find out if they were interested and how we should "plant" the "book". The library actually treated our boxes like any new acquisition. They are registered and have library security tags. The Library assigned each "book" it's own shelf number based on the Dewey Decimal System. Then Dave wrote up the clues so that the letterboxer has to use the resources of the Library to figure out the Dewey Decimal number of the book.
Originally the N.E.S.S. boxes were going to be a bit more spread out, but our own library here in town took the 3 that Dave had ready to show the Librarian.
It's a good idea to have a sample box ready to show, so that the Library knows what its getting itself into. We also gave the Desk a copy of the "What is Letterboxing" handout. Just in case someone comes upon the box by accident. A willing Library staff is key to this idea. Afterall it's their place.
