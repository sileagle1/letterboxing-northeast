Box report: Westborough and Shrewsbury, MA

From: cpascott | Date: 2004-09-21 13:37:48 UTC
We hunted all the boxes in Westborough and Shrewsbury, MA this past
Sunday. All were found, but a number were not well hidden. One was
sitting on top of the rock wall it was supposed to be under. We
noted that a number of the other boxes poorly hidden in addition to
this one fully exposed had recently been hunted by the same
letterboxers.

Rehide those boxes, people!

Boxes found:
Walkup Reservation
Peace on Earth <-- a great spot!
Time to go Fishin'
Bizarre Beetle
Crazy Caterpillar
Silly Spider
Idi Amin (Hitchhiker)
Table in the Woods
Flying Hippo
Bouncy Bug

Also, we're curious about the Idi Amin hitchhiker by Legerdemaine.
Is this part of a series of "Evil Dictators" HHs? We found it odd
that anyone would memoralize a man like Amin.
