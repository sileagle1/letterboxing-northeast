Journey into Imagination - Stafford CT

From: Shelley Bayer | Date: 2006-11-27 13:19:06 UTC
We started this series yesterday. There are 21 boxes. We had to drive almost an hour to get there. We didn't find the first box. The clues are hosted on a personal websirte so it doesn't say if they have been found or when and I wasn't given the option to contact the placer.

Anyway, we found the second so we kept going. The clues were well written and the boxes are close together which kept the kids' attention. We spent a lot of time on boxes 9 and 10 but never found them. We stopped after finding box 14, Strega Nona, because it was getting dark. We went on Sunday because there is not supposed to be hunting in the State forests on Sunday but we could hear a lot of shots. Happily, the boxes are all fairly close to a main trail, a few just a little way off it. We found out later there is a shooting range nearby so maybe it wasn't hunters, only target practice.

Anyway, this is a great series. We had read most of the books but not all and my kids want to read the ones they haven't.

We want to go back before it snows to continue from Strega Nona and find the last 7 boxes.

Shelley
momand2kids
