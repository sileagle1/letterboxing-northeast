The New Hampshire Tour Bus

From: team_new_hampshire | Date: 2007-05-28 03:42:17 UTC
So now that i had you all waiting, I think it is about time I let you
in on this box I have had in the works for sometime now. The New
Hampshire Tour Bus.

This is special box that moves around from location to location.
I have taken this idea from a box in texas by the same name.
I have decided to tweak it a little bit though.

WHAT YOU NEED TO KNOW:
====================
--This box is to picked up by the first person that finds it.

--What you should do as a possible finder, is carve a stamp of a
person or your choice to become a passenger of the tour bus.

--With a stamp carved, you would then go out and get this box, bring
it home, add your stamp to the Tour Bus box, and plan on replanting
it somewhere else as soon as possible. If you can not replant this
box somewhere new as soon as possible, then DON'T go looking for it.

--After picking up this box, you MUST post a message on the New
Hampshire messageboard on AQ, that you have picked up the box, then
repost when you have planted.

--After the box has been notified as picked up, I will then put the
box up for adoption for the finder to adopt and post the new clues
for the Tour Bus location. Everytime changing the stamp total to 1
number higher and leaving the name of the new passenger as well. This
process continues after everytime the box is reported as being taken
by a next first finder.

--The number of passengers can not exceed 20. So this means, try to
keep the stamp size reasonable. The Tour Bus is in a decent size lock
n' lock box, so we should be able to meet 20 passengers. If the box
has run out of room and will not hold 20 passengers, please post so,
the box will needed to be planted 1 more time by the last finder and
put the box up for adoption for me to adopt for the last time. The
Tour Bus will sit in this new location for others to find and not
pick up for 1 month before being retired and picking it up from that
location by myself. From there, the box will become something else
not yet determined.

--Ever stamp in the box is considered a find. The more stamps you
find in the box, the more stamps you can claim as being found. Please
do not let this discourage you from finding the box before anyone
else. You could always look for it later, when the bus is almost full
or resting at its final location.

--Also, if the logbook is full, please replace it with a new one,
preferably one just like the one in the box, but who am I to be picky.

--The difference between this Tour Bus and the one in Texas is, I am
allowing others from out of state to find this box. The only catch
is, I DO NOT want this box to EVER leave the state. Any out of
staters should look for this box with a new passenger in hand if you
choose to leave one, and replant at another loction in New Hampshire
before leaving the state. There is no obligation to take the box.

--This post will be listed on the box clues to look at when ever you
may need to.

--From what I can see, the only catch to this box is gas prices. You
must be aware that you might not find this box if you choose to go
looking for it. If someone goes looking for this box before you, it
may not be there when you get there. So if you decide to go looking
for this box, be prepared for the idea of coming home empty handed.

SOMETHING ELSE YOU NEED TO KNOW:
============================
The bus is ready to go, so get out there and come aboard.
Oh, and there is no bus driver, so if someone would like to carve the
bus driver, that would be awesome. I would have done it, but I
promised a friend that I would carve a stamps of her and make her a
passenger. I didn't want to fill up the box with my stamps, so if
someone wanted to do this, than again, that would be great.

If there are any questions, feel free to post them here and I can
hopefully answer them for you.
