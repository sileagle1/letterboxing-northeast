Everyone,

The response to the Hartford County Winter Drive has been overwhelming, and
that's very exciting! Here are some things to know if you are planning to
attend:

1. There have been some adjustments made to the route. Please eliminate
Tobacco Heritage, Cute Cluckers and Spring Has Sprung, due to some missing
reports. :/
However, you can add Inland Lobster in Windsor. :)

2. Mark and I have discussed the driving directions, and we don't think it's
necessary to make box-to-box directions. Please try to print yourself a set
of clues. There should be at least one set per car. We're going to caravan,
but in case someone gets seperated from the herd, every car should have a list
of where we're going.

3. There will be an event stamp. There will also be a stamp for us to use
to communally stamp into the boxes. The head count looks to be about 50, so
using one stamp to stamp in will probably speed up the process, along with the
fact that many of us already have a lot of these boxes. :)

4. In the case of snow/sleet/all that lovely New England stuff, Mark and I
will make the decision by 9 pm the night before. Please check Mark's page
(markandsuepepe.blogspot.com) to see if the drive is still on. So far, it looks
good!

Questions? Ask me. :) davyschris@...

Can't wait to see you all!

Chrissy
