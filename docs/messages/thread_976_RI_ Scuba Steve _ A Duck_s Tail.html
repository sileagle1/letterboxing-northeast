Letterboxing in the pitch dark this evening. (Extreme Letterboxing as my wife
calls it) Found "Scuba Steve" in Slater Park, Pawtucket, but had no
luck with "A Ducks Tail" in Diamond Hill Park, Cumberland. Has anyone found

Nightime letterboxing is a blast - wet shoes, a glow-in-the-dark compass,
flashlights and the "let's hope we don't run into the boogie man" heebie
jeebies...
