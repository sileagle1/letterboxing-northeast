Hey, guys. OK, so we have an issue on our hands here. I mean, AQ is great,
but we all know that there are lots of boxes that are only on LBNA, and while
we're still waiting for that to come up...

I have a bunch of clues on my hard drive for all around Connecticut. The
reason is that when I go to a town, I cut and paste all the clues onto one sheet
in order to save ink and paper when printing. I save these files because I
might want to print them out again next time I go. So I have a lot. I'm sure
other people have stashes as well.

I was thinking of posting some of these to this list. I'm not talking about
WOMs or anything. I'm talking about clues that, up until a few days ago, were
online for the whole world to see. What do you guys think? General
consensus?

In the meantime, if you're hoping to go to a particular place in CT, try
posting a cry for help here. I'm sure someone can get back to you with the clues
you're asking for. :)
