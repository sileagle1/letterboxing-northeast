I've found Grace's Hitchhiker twice this month, once in MA and once in
PA, but only counted the first find as an F. Only you can make the
decision as to whether you count yours once or three times, but in
general the first find is the only one most letterboxers count.

Silent Doug has some nice articles on PFX counts and hitchhikers here,
if you're looking for more info:

http://www.letterboxing.info/articles/00000003.php
