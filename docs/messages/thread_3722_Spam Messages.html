Some times the managers of web services, like Yahoo, react to waves
of spam by suddenly adjusting their filter rules. It will block the
spam, but sometimes legitimate messages get caught also. I have
heard of this happening with many ISP's, etc. Sometimes they will
adjust them back when the wave is over, and sometimes not.
